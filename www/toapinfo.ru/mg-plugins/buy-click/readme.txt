<p>Плагин предосталяет пользователю возможность быстрой покупки товара на сайте. Имеет страницу настроек для выбора необходимых данных от покупателя при быстрой покупке.</p>

<h2>Настройка плагина</h2>

<p>Для корректной работы плагина необходимо перейти в его настройки и указать необходымые параметры в опциях: поля формы заказа, способ доставки, способ оплаты.</p>

<p>В большинстве шаблонов плагин уже встроен и выводится без дополнительных действий. Если плагин у вас не выводится, то после его включения необходимо вставить шорт-код [buy-click id="<?php echo $data['id']?>" count="<?php echo $data['count']?>"] в файл шаблона product.php. Также возможно добавить кнопку в мини-карточке товара в каталоге, для этого необходимо добавить шорт-код [buy-click id="<?php echo $item['id']?>" count="<?php echo $item['count']?>" variant="<?php echo $item['variants']?>"] в файл шаблона catalog.php.</p>

<p>Внимание! Плагин не будет срабатывать, если в его опции выбрана доставка, для которой требуется выбирать адрес на карте, например СДЭК.</p>
<p>Внимание! Плагин не будет выводиться на странице товара, если у вас подключен плагин для минимальной суммы заказа, которая выше цены товара на странице.</p>
